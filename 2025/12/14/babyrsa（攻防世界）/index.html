babyrsa（攻防世界）

a1d729b93242433e86305169bf4765a9 (2).txt

题目描述

题目提供了以下信息：

  • 模数 n
  • 加密指数 e = 0x10001
  • 密文 c

服务器提供了一个”奇偶性预言机”（Parity Oracle）服务，当用户发送密文（十六进制格式）时，服务器会返回解密后明文的奇偶性（even或odd），然后立即断开连接。

目标是通过利用这个奇偶性预言机，恢复原始明文m，从而获取flag。

解题思路

RSA奇偶性攻击原理

RSA加密具有同态性质，即：

  • 如果密文c对应明文m，那么密文c’ = c × 2^e mod n对应的明文是m’ = 2m mod n

利用这个性质，我们可以构造一系列查询，通过分析2m mod n的奇偶性来推断m相对于n/2的大小关系：

  1. 如果2m mod n是偶数 → 2m < n → m < n/2
  2. 如果2m mod n是奇数 → 2m ≥ n → m ≥ n/2

通过多次这样的查询，我们可以使用二分搜索法逐步缩小m的可能范围，最终精确确定m的值。

攻击步骤

  1. 初始化参数：
    • 已知n、e、c
    • 设置初始搜索区间[0, n]
  2. 二分搜索攻击：
    • 对于每个比特位（共需log₂n次查询）：
      • 计算新密文c’ = c × (2^e)^k mod n（其中k是查询次数）
      • 发送c’给服务器，获取奇偶性
      • 根据奇偶性更新搜索区间：
        • 偶数：m在左半区间 → high = mid
        • 奇数：m在右半区间 → low = mid
  3. 恢复明文：
    • 搜索完成后，区间边界收敛到m
    • 将m转换为字节串，得到flag
