项目简介

这是一套专为CTF流量分析场景深度优化的自动化分析工具，基于Python开发，采用「自动化引擎做重活+多Agent做微推理」的架构设计，彻底摒弃了单一大模型包办一切的黑盒模式。

  • 90%的基础工作由自动化引擎完成，稳定、可复现、无幻觉
  • 10%的推理工作由多个微型Agent分工完成，每个AI只做一件极小的事，精准可控
  • 全链路每个环节都有完整的日志和输出，运维级可追踪、可调试
  • 支持在线Web可视化界面+本地命令行双模式，兼顾新手友好度和老手效率

现在你无需任何本地部署，直接访问上面的在线地址，就能使用全部功能！

核心亮点

1. 开箱即用，零门槛上手

无需安装Python、无需配置依赖、无需解决环境冲突，甚至不用注册登录。
只要打开浏览器访问在线地址，输入你的DeepSeek API Key，拖拽上传PCAP/PCAPNG流量包，一键就能启动全链路分析，哪怕是刚接触CTF的纯新手，也能在30秒内上手使用。

2. 运维级全链路流水线，全程可追溯

项目采用固定的8步标准化流水线，每一步的输入、输出、执行状态都完整记录，彻底告别黑盒分析：

哪怕分析出了问题，你也能精准定位到哪个环节出了错，甚至可以单独查看每个环节的输出结果，手动排查关键信息，所有数据都在界面上分标签页清晰展示。

3. 多Agent微推理架构，彻底根治AI幻觉

我们彻底放弃了「一个大模型搞定所有事」的模式，把AI的工作拆解到极致，每个Agent只负责一个极小的推理任务：

  • 记忆检索Agent：仅负责匹配历史相似题目，给出参考经验
  • 环节增强Agent：仅基于已提取的关键数据，给出解题方向
  • 防幻觉审查Agent：仅负责校验AI输出是否符合真实流量数据，杜绝编造内容

同时所有AI输出都以本地规则引擎的结果为最高优先级，本地已经扫描到的Flag，AI绝对不能篡改；本地没有的数据，AI绝对不能编造，从根源上解决幻觉问题。

4. 超强流量提取能力，不漏掉任何关键明文

针对CTF流量题的特性，我们做了深度优化的提取引擎：

  • 自动完成TCP流重组，把分段的数据包拼接成完整的会话内容，不会漏掉分段传输的Flag
  • 多编码兼容，支持UTF-8/GBK/GB2312/ASCII等多种编码，不会因为编码问题导致明文提取为空
  • 不过滤非打印字符之外的任何内容，哪怕是特殊格式的载荷，也会完整提取出来
  • 自动识别IP会话、HTTP请求/响应、TCP明文载荷，分类整理，一目了然

5. 全自动多层迭代解码，搞定套娃编码

CTF流量题最常见的考点就是多层编码套娃，比如两次URL编码、Base64+Hex混合编码，手动解码极其繁琐。
项目内置了全自动迭代解码引擎，支持URL/Base64/Hex/Unicode等CTF高频编码格式，自动循环解码，最多支持5层套娃，同时完整记录每一步的解码历史，你可以清晰看到明文的还原过程。

6. 完全开源免费，可扩展性拉满

项目完全基于Python开发，代码结构清晰，模块解耦，你可以轻松扩展新的解码格式、新的规则、新的Agent能力，适配更多CTF场景，甚至可以扩展到日常网络运维、应急响应的流量分析场景。
